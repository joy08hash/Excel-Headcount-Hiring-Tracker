# Headcount & Hiring Tracker

## Project Overview
An HR Analytics dashboard built in Microsoft Excel simulating a real-world recruitment operations tracker for a 20-role hiring pipeline across 7 departments.

## Tools Used
Microsoft Excel — Pivot Tables, VLOOKUP, COUNTIFS, AVERAGEIFS, Conditional Formatting, Charts

## What I Built
- Structured data model with 20 job records across 7 departments
- Pivot Table: Roles by Department
- Pivot Table: Average Days to Fill by Recruiter
- Bar Chart: Recruiter performance comparison
- VLOOKUP: Auto-pulls minimum salary from Lookup Table
- Summary Dashboard with 7 live KPI metrics
- Conditional formatting with color-coded status indicators

## Key Insights
- Engineering had the most open roles (3) — priority hiring area
- Linda Marsh filled roles fastest at 32.7 days
- Q1 achieved 88% fill rate vs 50% in Q2
